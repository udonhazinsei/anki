## 実行準備
1. [github](https://github.com/udonhazinsei/anki)開いて緑色の「Code」ボタンを押して一番下の「Download Zip」を押す
1. ダウンロードしたzipファイルを解凍
1. 解凍してできた「anki-main」フォルダをデスクトップに移動
1. 「anki-main」フォルダの中にankiでエクスポートしたファイルいれる

## 実行
ターミナルで
```
cd desktop/anki-main
```
```
python main.py
```
を順に実行
